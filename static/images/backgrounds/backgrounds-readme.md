# 决策魔方背景图片说明

本文档描述了决策魔方应用中各个页面所需的背景图片资源以及适用于AI图像生成的提示词。

## 图片存放位置

所有背景图片资源请统一存放在 `/static/images/backgrounds/` 目录下。

## 背景图片命名及规格

### 1. 首页背景

- **文件名**: `home_bg.png`
- **推荐内容**: 抽象几何图形或渐变色彩，与应用整体风格协调
- **推荐尺寸**: 1080 x 1920 像素 (全屏背景)
- **建议配色**: 应用主色调的延伸，柔和不影响前景内容
- **AI提示词**: `A minimalist abstract background with soft gradients in blue and purple tones. Subtle geometric patterns with low opacity. Clean, modern mobile app background. No text or prominent elements. Vertical orientation 9:16 ratio.`

## 图片格式要求

- **文件格式**: 优先使用PNG格式，支持透明度
- **分辨率**: 建议使用2倍分辨率（@2x）以适应高分辨率屏幕
- **文件大小**: 每张图片建议控制在500KB以内，保证应用加载速度
- **压缩**: 建议使用有损压缩以减小文件大小，同时保持视觉效果

## AI生成图片建议

1. 使用以上提供的提示词向AI图像生成模型请求背景图片
2. 确保指定垂直方向的9:16比例，适合移动设备屏幕
3. 可以根据需要调整提示词，但保持核心视觉元素和色调
4. 生成多个版本并选择最适合的一个作为最终背景
5. 检查生成的图像是否有明显的人工痕迹或不自然元素，必要时重新生成
6. 确保背景足够简洁，不会干扰UI元素和文本内容

## 图片缺失处理

如果对应的背景图片文件缺失，系统将使用该页面或主题的默认渐变色背景作为替代，确保用户体验不会受到影响。但为了获得最佳效果，建议提供所有页面和主题的背景图片资源。

## 使用建议

1. 背景图片应当保持低调，不应喧宾夺主，干扰用户操作和内容阅读
2. 考虑在背景图上添加半透明遮罩层，以确保前景内容可见度
3. 各主题的背景应保持一致的风格，提供连贯的用户体验
